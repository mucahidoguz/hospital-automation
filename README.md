# HASTANE RANDEVU OTOMASYON

Hastaların listenmesi, randevularının takibi uygulanan tedavilerin ve verilen ilaçların görüntülenmesi üzerine bir otomasyon projesi üzerinde çalışılmıştır.


## KULLANILAN 

-JSON Server
-React
-React Router Dom
-Redux
-Axios
-Material UI (CSS)
